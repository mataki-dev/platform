# Security Policy

## Reporting a Vulnerability

**Please do not open public issues for security vulnerabilities.**

Report vulnerabilities via GitHub private security advisories:
https://github.com/mataki-dev/platform/security/advisories/new

We aim to acknowledge reports within 3 business days.

## Supported Versions

The latest minor release on `main` is supported. Older versions are not
backported; upgrade to the latest release to receive security fixes.
